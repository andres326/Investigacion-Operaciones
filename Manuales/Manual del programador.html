Manual del programador

1. Introducción

Aquí se dará a conocer la estructura que tiene la aplicación, explicadose la forma en la que se menaj las variables, las claes y los paquetes; para que de esta forma el programador pueda acceder a la aplicació correctamente y entienda la forma en que se manejó la estructura de la misma.

Para inciar se mencionarán los paquetes que componen el programa y esplicar su fin. Luego de ello se explicara funcionamiento de cada clase respectiva composición en cuanto a variables y metodos.

 2. Instalación

Para iniciar el programa se debe tener instalado el IDE para Java llamado NetBeans 8.0.2, al iniciarlo se abre el proyecto que se encuentra en la carpeta IOFinal.

De igual forma se encuentra la librería JFreeChart y la librería JCommon en la carpeta librerías, ellas deben ser instaladas desde el IDE de la siguiente forma: El el proyecto dar click derecho en la carpta llamada "Libraries" y escoger la opción que dice "add JAR/Folder". Y buscar la carpeta donde se encuentra la libería.

 3. Paquetes

Los paquetes que se encuentran en la aplicación se tienen con el fin de poder separar de forma ordenada las clases para que se pueda tener un buen entenimiento de la estructura que tiene la aplicación.

Los siguientes son los paquetes que se encuentran en el programa:
* Main
* PL
* Vista
3.1 Paquete Main

Este paquete se encarga de contener únicamente la clase Laucher.

3.2 Paquete PL

El nombre del paquete se debe a que contiene todas las clases que se encargan de la lógica central del programa. Ellas hacen parte de la programación lineal que maneja la aplicación.
3.3 Paquete Vista

Éste paquete contiene todas las clases que forman la parte grafica, que es la que permite el contacto con el usuario y el manejo como tal del programa.
